# vue3-ts-vite

_Template for develop vue3 projects using typescript and vite in docker_

## Begining 🚀

### Requeriments 📋

\_docker & docker-compose.

### Deployment 📦

```sh
git clone git@github.com:leogp/vue3-base-app.git

cd ./vue3-base-app

docker-compose up --build
```
